
primero activar un entorno virtual en la carpeta raiz "backend-python"
esto es para windows para linux verificar: https://flask.palletsprojects.com/es/main/installation/
py -3 -m venv .venv
luego activarlo
.venv\Scripts\activate

ya activado se descargan las dependencias
pip install Flask
pip install python-dotenv
pip install flask-cors
pip install bcrypt
pip install mysql-connector-python
pip install Blueprint

archivo .env va en la raiz de la carpeta backend-python antes de src

ya en produccion pip install gunicorn y con (.env) activo en linea de comandos

gunicorn --bind 0.0.0.0:5000 wsgi:app
